# Научиться учиться

## Описание проекта
- Проект представляет собой веб-страницу, содержащую информацию о таких эффективных методах обучения, как методы Барбары Оакли, метод фейнмана, методы Джоша Кауфмана
- Использованы технологии HTML5, CSS
